# Quote Generator

A beautiful and interactive web application that generates random inspirational quotes with text-to-speech and copy functionality.

## Features

- **Random Quote Generation**: Get new inspirational quotes with a single click
- **Text-to-Speech**: Listen to quotes using the speech functionality
- **Copy to Clipboard**: Easily copy quotes to share with others
- **Beautiful Design**: Clean and modern user interface with elegant typography
- **Responsive Layout**: Works perfectly on all device sizes
- **Visual Feedback**: Success message when quote is copied
- **Author Attribution**: Display the author of each quote

## Technologies Used

- **HTML5** - Semantic structure and accessibility
- **CSS3** - Modern styling with flexbox and animations
- **JavaScript** - Dynamic quote generation and interactive features
- **Google Fonts** - Poppins and Open Sans typography
- **Font Awesome** - Comprehensive icon library
- **Web Speech API** - Text-to-speech functionality
- **Clipboard API** - Copy to clipboard feature
